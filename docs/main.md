# ccme-dispenser 项目要求

## 项目目的
本项目旨在省去繁琐的溶液装配步骤，将在自动化程序的帮助下，实现固体溶质溶解，以及溶液小瓶装配的无人化

## 外设一览
[+] 蠕动泵两个
[+] FMC控制器操控的三轴系统
[+] ika 可加热磁力搅拌
[+] OV摄像头
+ 香橙派5 pro

## 项目要求
[+] 事先将装有试剂的烧杯放于磁力搅拌上，这是唯一人类操控的一步
[+] 该项目编写的后端程序将运行在装有Debian 系统的香橙派上，环境为aarch64
[+] 人类将通过443端口https服务访问前端，返回控制面板网页
[+] 网页将向后端发送相关信息，包括需要注入的清水体积，此时程序将操控蠕动泵1运动注入清水
[+] 开启搅拌
[+] 摄像头已经事先对准溶液正上方，接着程序需要通过摄像头判断溶解是否完全
[+] 直到溶解完全后，停止搅拌，需要程序操控移动三轴系统至指定位置，然后启动蠕动泵2将溶液运送至指定位置
[+] 重复上一步直到溶液完全分发完毕

## 说明
[+] 外设的有关说明均在本目录以及子目录中，必须阅读完所有文件

## 程序要求
[+] 要求前后端分离完全，前端采用html+css+javascript编写，目录在frontend
[+] 后端采用C++23标准编写，采用google标准，使用expected, optional 等, 禁用异常
[+] 前端流量由nginx托管，然后反向代理给后端，前后端之间采用http连接，无需tls二次验证
[+] 后端各个模块之间请分开编写程序，之间采用d-bus协议通信
[+] 所需模块可通过git拉取到3rdparty目录
